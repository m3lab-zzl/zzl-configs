# zzl-configs

- installation

  - ```sh
    cd 
    mv zzl-configs{,.bak} 
    git clone git@gitee.com:deelin-zzl/zzl-configs.git
    # you may change the folder name, and cd to the new name
    cd zzl-configs
    bash ins.sh
    ```
